# Introduct 
- Là một float Toast (thông báo nổi) được sử dụng để thông báo trạng thái của một action 

Example:
```js
<ToastContainer

	position="top-right"
	autoClose={5000}
	hideProgressBar={false}
	newestOnTop={false}
	closeOnClick
	rtl={false}
	pauseOnFocusLoss
	draggable
	pauseOnHover
	theme="light"
	transition: Bounce,
/>
{/* Same as */}
<ToastContainer />
```
